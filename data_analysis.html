Data Analysis

  • Understanding the Dataset
  • Do ETL (Extract, Transform, Load)
  • Exploratory Data Analysis
  • Conclusion
Data Collection:
This process involve gathering of right and correct dataset to do the analysis. Now your data might be in a database, you will need to write sql query to extract the data from the database for the analysis, export the extract data in csv or xlsx format to open in Microsoft Excel, then come up with pivot table to create a dashboard.
You can open the csv or xlsx format also in Tableau for great visualization, I have many tableau project under tableau section
Data can also be download from free website online: Kaggle.com, makeovermonday.co.uk, data.world.com etc
Data can be purchase from vendor as well

Data Cleaning:
This process invlove removing missing data or fill in the missing data, also removing duplicated values from your dataset

Exploratory Data Analysis (EDA):
This process involve ask logical questions and solving it through visualizations from your dataset, to gain insight into the dataset.

Creating Dashboard:
This process involve putting together all the charts(eg pie, bar histogram, custom chart) to tell your findings in the data along with the Key Performance Indicators (KPIs)

Presentation:
This process involve presenting your dashboard to the management with suggestions on the data towards improvements and then another dashboard can be made based on management decision for trends in the data etc....
